# LLM API Project (Python)

This project demonstrates how to call an LLM API from Python, use a tool (function calling), and return the result back to the model.

## 🚀 Features

- Calls OpenAI LLM API from Python
- Demonstrates tool/function calling
- Executes a custom calculation function
- Sends tool output back to the model
- Prints final response

---

## 🛠 Technologies

- Python
- OpenAI API
- python-dotenv
